gitw - Git 命令包装器，生成 git 变更记录日志

golanggookitgit

gitw - Git 命令包装器，生成 git 变更记录日志，获取 repo 信息和一些 git 命令工具。

  • 包装本地 git 命令
  • 快速运行 git 命令
  • 快速查询存储库信息
    • 获取 remote, branch 详细信息
  • 通过 git log 快速生成版本变更日志
    • 允许自定义生成配置
    • 允许自定义生成过滤、样式等
    • 可以直接在 GitHub Actions 中使用
