通过 scoop 在Windows 下安装工具

  • 10 7月 2022
  • •1 分钟阅读

scoop 是一个Windows 下的应用包安装管理工具，类似于 Linux 下的 yum,apt 和MacOs 下的 brew。

通过使用 scoop 可以方便快速的安装常用的开发工具，会自动设置环境变量，例如 php, go, rust 等。

Scoop github: https://github.com/ScoopInstaller/Scoop

安装Scoop

iwr -useb get.scoop.sh | iex

添加bucket 支持更多工具:

scoop bucket add extras

基本使用

查看可用命令：

scoop

搜索工具

scoop search git
scoop search php
scoop search go

输出类似：

安装工具

scoop install git

卸载软件

scoop uninstall git

更新scoop及软件

scoop update #更新 scoop 自己
scoop update git #更新 git

安装aria2加速下载

scoop安装aria2后，scoop会自动调用aria2进行多线程下载以加速下载：

scoop install aria2
# 关闭每次运行时的告警提示
scoop config aria2-warning-enabled false

安装常用工具

基础工具

scoop search git

语言环境

scoop search php
scoop search go
scoop search nodejs

其他工具

scoop install composer
scoop install yarn
