Show HN: eget - install GitHub release binaries

## url

## text

I built eget, a small CLI for installing prebuilt release binaries directly from GitHub releases, GitLab/Gitea, SourceForge, direct URLs, and internal download templates.

The main use case is trying new tool versions before central package indexes catch up.

Blog: https://inhere.github.io/en/blog/2026/eget-intro/
