Show HN: miglite - raw SQL migrations for Go

## url

## text

I built miglite, a small raw-SQL migration tool for Go.

It keeps migration files as plain SQL with UP/DOWN sections, supports DATABASE_URL and multiple migration directories, and avoids bundled DB driver dependencies when used as a library.

Blog: https://inhere.github.io/en/blog/2026/gookit-miglite-intro/
